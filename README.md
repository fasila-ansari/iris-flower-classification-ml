# iris-flower-classification-ml
Machine learning project that classifies iris flower species using the Scikit-learn library in Python
## Overview
This project uses a machine learning classification algorithm to predict the species of an iris flower based on its physical characteristics such as sepal length, sepal width, petal length, and petal width.

The model is trained using the well-known Iris dataset provided by the Scikit-learn library.
## Technologies Used
- Python
- Scikit-learn
- Pandas
## How It Works
The program loads the Iris dataset and trains a machine learning model using the K-Nearest Neighbors (KNN) algorithm.
The user enters the flower measurements, and the model predicts the species of the iris flower.
## Example
Input:
Sepal Length: 5.1
Sepal Width: 3.5
Petal Length: 1.4
Petal Width: 0.2
Output:
Predicted Species: Setosa
## Purpose
This project was created to practice basic machine learning classification techniques and gain hands-on experience with Python and Scikit-learn.
